Step 1: Install TamperMonkey

TamperMonkey is a browser extension that allows you to run custom scripts on websites.

  • Chrome: Chrome Web Store
  • Firefox: Firefox Add-ons
  • Other browsers: TamperMonkey.net

Step 2: Get an AI API Key

You'll need an API key from one of these providers (see comparison below):

  • Google Gemini: Get Gemini API Key (Recommended - 1,500 requests/day FREE)
  • Groq: Groq.com (1,000 requests/day, super fast)
  • OpenRouter: OpenRouter.ai ($5 free credits)

Step 3: Install the Script

  1. Download the ixl-auto-answer-gemini.user.js file from this project
  2. TamperMonkey will automatically detect it and prompt you to install
  3. Click "Install" in the TamperMonkey dialog

Step 4: Configure the Script

  1. Visit any IXL.com page
  2. You'll see the script panel in the top-right corner
  3. Click "Toggle Settings"
  4. Paste your API key and save
  5. Click "Test Key" to verify it works

Q: Why am I hitting rate limits after only 20 questions?

A: You were hitting the per-minute limit, not the daily limit. Here's what happened:

  • Old version sent images + HTML (very large requests)
  • Gemini 1.5 Flash: 15 requests/minute limit
  • Gemini 2.0 Flash: 10 requests/minute limit
  • If you did 20 questions in 2 minutes, you hit the limit!

‚úÖ Solution: Version 14.0 sends HTML-only (no images), reducing request size by 50-70%. You can now pace yourself to stay under 15 requests/minute, or wait a minute between batches.
